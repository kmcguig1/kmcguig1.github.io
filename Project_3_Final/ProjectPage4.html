INTERACTIVE LEARNING GAME

Web Design Project

Tasked with creating an interactive web learning game, I designed a game for elementary and middle school-aged children to learn about the world and traveling.

The home screen gives options of six different locations in the world for the user to learn about before deciding which place they would like to travel to. After choosing, the user is brought to a screen with a checklist of items to bring and activities to complete before boarding the plane. By clicking on each item, the character completes the task until the user is ready to take off to their chosen location.
